## Prerequisites
- Install the .NET SDK (version 8.0; download from https://dotnet.microsoft.com/download). Verify with `dotnet --version` in a terminal.
- VS Code

## Running the Code
Open the code in VS Code then run the following commands in the integrated terminal:
- Build: type `dotnet build` (compiles to binaries; check for errors).
- Run: type `dotnet run` (outputs 5 sentences to console).
